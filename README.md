## 色采是什么？

简单说，色采是一款帮你「取色 + 配色 + 管理色卡」的手机工具，支持 iOS 和 Android 双平台。

它最让人印象深刻的功能，是**相机实时配色**：打开 App，对准任何你觉得好看的东西——咖啡桌、窗外的天色、一件衣服——镜头扫过去就自动提取色彩，生成色卡。就是那种「诶，我刚才看到一个特别好看的土橙色，赶紧拍下来」的场景。

除了实时取色，色采还有：

- **照片智能配色**：从相册里的照片里提取颜色，算法识别配色方案；
- **灵感探索**：内置 800+ 张原创色卡，来源涵盖西方油画、故宫藏品、电影动画、自然风光等等，每天还有推荐；
- **精准调节**：支持 HSB / HEX / CMYK / RGB 四种色值模式，手动微调；
- **配色方案推荐**：互补色、近似色、单色、混合色，一键生成方案；
- **分享 & 联动**：可以直接导入 Procreate、Adobe，和番茄清单、Memory Helper 等 App 联动。

App Store 有多个官方专题推荐过它，也被少数派、小众软件、最美应用等媒体写过，在全球 100 多个国家都有用户。

## 免费版 vs 高级版，差距大吗？

这是很多人纠结的点。色采的免费版功能其实挺完整的，相机取色、照片配色这些核心功能免费就能用。

但高级版（付费版）解锁的东西也挺实在：

| 功能 | 免费版 | 高级会员版 |
|------|--------|------------|
| 原创色卡 | 部分色卡 | 800+ 张全部解锁 |
| 随机颜色次数 | 有限次数 | 无限次数 |
| 相机实时配色 | ✓ | ✓ |
| 照片智能配色 | ✓ | ✓ |
| 数据云端备份 | — | ✓ |
| 配色方案推荐 | — | ✓ |
| 所有随机主题 | — | ✓ |
| 原创色卡商业授权 | — | ✓ |
| 定制色卡分享 LOGO | — | ✓（仅安卓端） |

如果你只是偶尔找找配色灵感，免费版够用。但如果你是做设计的、或者有商业使用需求，高级版值得入手，尤其是**商业授权**这一点很关键——免费版的色卡是不能用于商业项目的。

## 关于 iOS 和安卓的数据互通

这里有一点需要注意：**iOS 版和安卓版支持数据互通，但不是自动同步的**，需要通过「备份和恢复」手动操作。

而且，如果你在 iOS 端选择了用 iCloud 备份，那就没法和安卓端同步了——两种备份方式二选一。双平台用户需要留意这个细节。
